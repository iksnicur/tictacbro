Tic Tac Bro v0.1



ABOUT:
------------------
Tic Tac Bro is a web application made by Justin Rucinski
to demonstrate a simple tic-tac-toe game using AI that
will always end in a cat's game.

After teaching myself (Justin Rucinski) some Javascript and Node.js
I decided to create this simple tic-tac-toe game to start a coding
portfolio for myself to demonstrate some of my capibilties.

Any/All feedback is welcome!
